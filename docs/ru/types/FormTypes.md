# FormTypes

`FormTypes` - содержит константы для всех возможных типов полей ввода HTML форм, определенных в спецификации HTML5. Этот класс предоставляет централизованное и типобезопасное хранилище для типов элементов форм, что упрощает разработку и предотвращает ошибки при использовании строковых литералов.

---

## Импорт

```python
# Импорт всего класса
from LayoutML.FormType import FormTypes

# Использование констант
text_field = FormElement(form_type=FormTypes.TEXT)
email_field = FormElement(form_type=FormTypes.EMAIL)
submit_button = FormElement(form_type=FormTypes.SUBMIT)
```

## Особенности каждого типа

### Особенности текстовых полей

#### FormTypes.TEXT

- Базовый тип для любого текста
- Рекомендуется использовать `maxlength` и `minlength` для валидации
- Поддерживает `pattern` для регулярных выражений

#### FormTypes.EMAIL

- Автоматическая валидация формата `email`
- На мобильных устройствах показывает специальную клавиатуру с @
- Рекомендуется использовать `autocomplete="email"`

#### FormTypes.PASSWORD

- Символы заменяются точками или звездочками
- Используйте `autocomplete="new-password"` для новых паролей
- Используйте `autocomplete="current-password"` для существующих

#### FormTypes.NUMBER

- Только числовой ввод
- Используйте `min`, `max` и `step` для ограничений
- На мобильных показывает цифровую клавиатуру

### Особенности элементов выбора

#### FormTypes.CHECKBOX

- Можно выбрать несколько вариантов
- Требует атрибут `value`
- Используйте `checked` для предварительного выбора

#### FormTypes.RADIO

- Только один выбор в группе (одинаковый name)
- Требует атрибут `value`
- Используйте `checked` для выбора по умолчанию

#### FormTypes.SELECT

- Для создания выпадающих списков
- Используйте `multiple` для множественного выбора
- Используйте `size` для указания количества видимых строк

### Особенности специальных полей

#### FormTypes.DATE, FormTypes.TIME и другие временные типы

- В современных браузерах показывают специальный интерфейс выбора
- Формат значений зависит от типа
- Используйте `min` и `max` для ограничения диапазона

#### FormTypes.FILE

- Открывает диалог выбора файлов
- Используйте `accept` для ограничения типов файлов
- Используйте `multiple` для выбора нескольких файлов
- Используйте `capture` для доступа к камере на мобильных

#### FormTypes.RANGE

- Создает ползунок для выбора значений
- Используйте `min`, `max` и `step` для настройки
- Не показывает текущее значение (требуется JavaScript)

#### Особенности кнопок

#### FormTypes.SUBMIT

- Отправляет форму
- Может иметь атрибуты `formaction`, `formenctype`, `formmethod`, `formtarget`
- По умолчанию валидирует форму перед отправкой

#### FormTypes.IMAGE

- Кнопка в виде изображения
- Обязательные атрибуты: `src` и `alt`
- Отправляет координаты клика (x, y)
