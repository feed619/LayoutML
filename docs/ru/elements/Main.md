# Main

`Main` - это специализированный класс для создания семантического HTML элемента <main>. Класс наследуется от [BaseElement](../base/BaseElement.md) и предназначен для обозначения основного, уникального содержимого документа, которое не повторяется на других страницах сайта (в отличие от шапки, подвала, навигации и боковых панелей).

---

## Импорт

```python
from layoutML.elements import Main
```

## Наследование

- Родительский класс: BaseElement
- Тип элемента: main (не самозакрывающийся тег)
- Назначение: Обёртка для основного уникального контента страницы

## Семантическое значение

Элемент <main> представляет основное содержимое документа или приложения. На странице должен быть только один элемент <main>, и он не должен содержать контент, который повторяется на других страницах (например, боковые панели, навигацию, информацию об авторских правах, логотипы сайта и формы поиска).

## Конструктор

### **init**(object_name=None, style=None, boolean_attributes=[], \*\*kwargs)

Создаёт новый элемент основного содержимого с указанными параметрами.

Параметры:

- object_name (опционально): Имя/идентификатор элемента
- style (опционально): CSS строка для inline стилей
- boolean_attributes (опционально): Список булевых атрибутов
- \*_kwargs: Дополнительные HTML атрибуты (id, class\_, aria-_ и т.д.)

### Автоматически устанавливаемые свойства:

- Тег: main
- self_closing: False (не самозакрывающийся тег)
- object_type: "MainElement"

### Примеры:

```python
# Простой элемент main
main = Main()
# Main с именем и классами

main = Main(
object*name="mainContent",
class*="content-wrapper",
id="pageMain"
)
# Main с inline стилями
main = Main(
style="max-width: 1200px; margin: 0 auto; padding: 20px;",
aria_label="Основное содержимое страницы"
)
```

## Атрибуты класса

| Атрибут      | Тип  | Описание Значение по умолчанию                    |
| ------------ | ---- | ------------------------------------------------- |
| object_type  | str  | Тип объекта (всегда "MainElement") "MainElement"  |
| tag          | str  | HTML тег (всегда "main") "main"                   |
| self_closing | bool | Флаг самозакрывающегося тега (всегда False) False |
